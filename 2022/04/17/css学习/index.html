  • 属性：

    • border-radius：将图片四个角定义为圆弧状，50%为圆
    • margin-bottom：外边距，表示每个标签 后面预留 多少空间
    • transform：scale() ——将其比例改变
    • transition : 改变时间，即渐变时间
    • color：字体颜色
    • background-color：背景颜色
    • opacity: 透明度

  • 同一个页面内的id属性应该是唯一的

  • css 三种定义方法： 行内样式表 ， 内部样式表，外部样式表

    ​ 内部： <style type="text/css" ></style>

    ​ 外部： <linK rel= "stylesheet" href="xx" type="text/css" >

  • 同一个标签 class 可以有多个，使用空格隔开即可,

  • 选择器：

    • 元素（标签）选择器：
    • id选择器: #id {}
    • 类选择器： .class{} ，对颜色的作用，根据css定义的最后一个改变，与class 定义顺序无关

  • 伪类选择器： 可理解为状态标签，返回布尔值判断是否符合要求

    • :link —— 链接未访问前的状态
    • :visited ——链接点击后的转态
    • :active ——鼠标长按点击时的状态
    • :hover ——鼠标当鼠标悬停时的状态，样式改变
    • :focus —— 聚焦后的状态 ， 如：输入框

  • 位置伪类选择器

    • :nth-child(n) ——选择是其父标签第n个子元素的所有元素，n可以为布尔式子

      n 可取 odd 所有偶数的 ，even所有奇数的， 3n以3为倍数

  • 目标伪类选择器

    • :target: 当 url 指向时生效

  • 复合选择器：

    • 将多个选择器以 逗号隔开使用

  • 通配选择器：

  • 伪元素选择器

文本

  • text-align : 文字对齐方式 ， left rignt center justify 行内内容对齐，子标签可继承父标签
  • line-height: 行高可以让内容竖直居中
  • letter-spacing: 字间距
  • text-indent: 首行缩进
  • text-decoration: 文本修饰 ， 下划线、删除线等
  • text-shadow: 文字阴影 ，有偏移坐标text-shadow: 5px 2px 2px green; 分别为 x ，y 方向偏移量 ， 模糊值 和颜色。可以有任意多个阴影 使用 ， css表示并列成分

字体

  • font-size：字体大小
  • font-style: 斜体，字体样式
  • font-weight: 粗细
  • font-family：字体种类

背景

  • background-color:

  • background-image:

  • background-size: 属性 cover 覆盖原则

  • background-repeat:是否重复

  • background-position： 初始位置

  • background-attachment: 背景相对于其中得文本变化

    fix ：固定浏览器视图， local：背景与文本相对应,若内容移动与之相对的背景会随着文本移动，内容和背景从内部到外部的表现都是相对应的，两者共同移动

    scroll : 背景与包含内容对应作为整体 ，但若内容移动，其不会改变 ，外部表现为一致改变，内部元素移动，背景不会改变，两者内部相对移动，外部一致移动

  • border-collapse ： 将分离边框重合在一起
